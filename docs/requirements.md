### Non-functional Requirements

Possible non-functional requirements for the workflow include:

1. The project shall run without problems given that all required are installed properly.
2. The total execution time of the whole project (From reading the raw data to outputting the final output) shall be executed under a minute.
3. The whole project shall be idempotent, running the project should produce identical output results and visualizations.
4. The project shall support adding more data given they are available.
